Privacy Policy

DA1HAE Studio는 서비스 문의, 마케팅 운영, Influencer CRM 베타 운영을 위해 필요한 최소한의 정보를 처리합니다.

수집하는 정보

  • 문의 시 제공하는 이름, 브랜드명, 이메일, 연락처, 상담 내용
  • Influencer CRM 사용자가 직접 저장한 인플루언서 후보 정보, 태그, 메모, 상태값
  • 사용자가 연결한 Instagram Professional 계정의 허용된 데이터 범위 안에서 제공되는 게시물 및 댓글 반응 관련 데이터

이용 목적

  • 문의 응대 및 서비스 안내
  • 마케팅 서비스 제공 및 커뮤니케이션
  • Influencer CRM 기능 제공, 캠페인 운영 기록, 발행 콘텐츠 확인

Instagram API 데이터 사용

Influencer CRM은 사용자가 직접 저장한 후보 정보를 정리하고, 연결된 Instagram Professional 계정의 허용된 데이터 범위 안에서 운영 관리를 돕는 도구입니다. 승인되지 않은 데이터 수집이나 자동화된 대량 메시지 발송 기능은 제공하지 않습니다.

보관 및 삭제

정보는 서비스 제공 목적에 필요한 기간 동안 보관하며, 사용자가 삭제를 요청하면 확인 후 합리적인 기간 내 삭제합니다. 삭제 요청은 ramy_da@naver.com으로 보낼 수 있습니다.

데이터 삭제 요청 안내

DA1HAE Studio 및 Influencer CRM에서 처리되는 문의 정보, 서비스 이용 정보, Instagram 연결 데이터 삭제를 원하시는 경우 이메일로 요청할 수 있습니다.

삭제 요청 방법

  • ramy_da@naver.com 으로 삭제 요청 메일 발송
  • 메일 제목: Data Deletion Request
  • 포함 정보: 이름 또는 브랜드명, 문의 시 사용한 이메일, 연결한 계정 또는 식별 가능한 정보, 삭제를 원하는 데이터 범위

처리 절차

요청 내용을 확인한 뒤 본인 또는 권한 있는 사용자의 요청인지 검토하고, 합리적인 기간 내 관련 데이터를 삭제하거나 비식별 처리합니다.

Instagram 연결 데이터

Influencer CRM은 사용자가 연결한 Instagram Professional 계정의 허용된 데이터 범위 안에서만 데이터를 사용합니다. 연결 해제 또는 삭제 요청 시 서비스 제공에 필요한 관련 데이터를 삭제합니다.
