Meta Review

Meta Review Information

이 페이지는 DA1HAE Studio가 준비 중인 Influencer CRM의 Meta Business Verification 및 Instagram API App Review를 위해 서비스 목적과 데이터 사용 범위를 설명합니다.

서비스 배경

Influencer CRM은 인플루언서 협업 후보, 진행 상태, 발행 일정, 게시물 링크, 댓글 반응을 캠페인 단위로 정리하기 위해 준비 중인 운영 도구입니다.

서비스 목적

Influencer CRM은 브랜드와 마케팅 담당자가 인플루언서 협업 후보, 태그, 상태값, 담당자 메모, 발행 일정, 게시물 링크, 댓글 이벤트 데이터, 댓글 반응 분석을 정리하는 운영 도구입니다.

Instagram API 사용 목적

사용자가 연결한 Instagram Professional 계정의 허용된 데이터 범위 안에서 발행 콘텐츠 확인, 게시물 링크 기록, 댓글 이벤트 데이터 정리, 발행 콘텐츠 댓글 반응 분석을 지원하기 위해 사용합니다.

Compliance

This service does not provide or support automated bulk DM sending, unauthorized scraping, or the sale of personal data. It is not intended to bypass Instagram or Meta platform policies.

주요 기능 범위

  • 인플루언서 후보 저장 및 분류
  • 태그, 카테고리, 상태값 관리
  • 담당자 및 내부 메모 관리
  • 발행 일정 캘린더
  • 게시물 링크 및 업로드 현황 기록
  • 댓글 이벤트 데이터 정리
  • 발행 콘텐츠 댓글 반응 분석
  • CSV/XLSX 다운로드
